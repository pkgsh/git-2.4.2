# pkgsh/git-2.4.2

This package will install Git 2.4.2 from source.

### Installation

```
install.sh pkgsh/git-2.4.2
```
or

```
sh <(curl -sL http://install.opensource.sh/pkgsh/git-2.4.2)
```
